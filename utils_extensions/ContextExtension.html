ContextExtension

ContextExtension extension

An extension on BuildContext that provides convenience methods for obtaining fractional heights and widths based on the screen size.

This extension allows quick access to screen dimensions as a fraction of the total height or width, which can be useful for responsive layouts.

Example usage:

// Get half the screen height
double halfHeight = context.heightFraction(sizeFraction: 0.5);

// Get a third of the screen width
double thirdWidth = context.widthFraction(sizeFraction: 0.33);
on
  • BuildContext

Methods

heightFraction({double sizeFraction = 1}) → double
Returns a fraction of the screen height.
widthFraction({double sizeFraction = 1}) → double
Returns a fraction of the screen width.
